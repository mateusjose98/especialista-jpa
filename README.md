# Especialista JPA - Alga Works Treinamentos


## O que é JPA:

- O JPA (Java Persistence API - Jakarta Persistence) é uma especificação oficial do mundo Java que tem por objetivo de padronizar os ORMs que a implemente. Obviamente, ele precisa de algum item que o implemente, os mais famosos hoje são Hibernate, EclipseLink e OpenJPA.
